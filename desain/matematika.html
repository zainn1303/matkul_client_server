Pengertian Logaritma

Logaritma adalah kebalikan dari suatu perpangkatan. Jika sebuah perpangkatan ac = b, maka dapat dinyatakan dalam logaritma sebagai: alog b = c dengan syarat a > 0 dan a \ne 1

Pada penulisan logaritma alog b = c, a disebut bilangan pokok dan b disebut bilangan numerus atau bilangan yang dicari nilai logaritmanya (b > 0) dan c merupakan hasil logaritma. Jika nilai a sama dengan 10, biasanya 10 tidak dituliskan sehingga menjadi log b = c. Jika nilai bilangan pokoknya merupakan bilangan e (bilangan eurel) dengan e = 2,718281828 maka logaritmanya ditulis dengan logaritma natural dan penulisannya dapat disingkat menjadi ln, misalnya elog b = c menjadi: ln b = c. Perpangkatan Contoh Logaritma
21 = 2 2log 2 = 1
20 = 1 2log 1 = 0
23 = 8 2Log 8 = 3
